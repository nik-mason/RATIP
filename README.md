# 🗺️ Ratip: 나만의 지도를 그리는 여정

**Ratip**은 단순한 맛집 검색 앱이 아닙니다. 남들의 입맛과 취향에 휘둘리지 않고, 오직 **나만의 기준**으로 세상을 기록하고 탐험하는 **개인화 위치 기록 & 게이미피케이션 플랫폼**입니다.

### 🌟 왜 Ratip인가요?
세상에는 수많은 리뷰가 있지만, 정작 나에게 딱 맞는 장소를 찾기는 어렵습니다. Ratip은 "남들이 좋다는 곳"이 아니라 **"내가 가본 곳, 내가 좋아할 곳"**에 집중합니다. 내가 발을 딛는 모든 곳이 나만의 업적이 되고, 내가 남기는 한 줄의 평이 나만의 데이터가 되어, 세상에 단 하나뿐인 나만의 지도를 완성해 나갑니다.

### ✨ 핵심 가치
1. **진실한 기록**: 반드시 해당 장소에 **실제로 방문**해야만 리뷰를 남길 수 있습니다. 📍
2. **나만의 기준**: 타인의 시선에서 자유로운 **'나만 보기'**가 기본 설정입니다. 🔒
3. **성장의 재미**: 30단계의 랭크와 50개의 업적을 달성하며 **성장하는 RPG**의 재미를 제공합니다. 🏆
4. **축제의 연속**: 매달 1일과 15일, 특별한 미션과 보상이 쏟아지는 **정기 이벤트**가 열립니다. 🎁

## 🚀 주요 기능

### 1. 🔐 보안 로그인 (Supabase Auth)
- 이메일/소셜 로그인 지원
- **Face ID / 지문 인식** 연동.

### 2. 🗺️ 나만의 맞춤 지도 (Naver Map)
- 사용자의 실시간 위치 중심 고정밀 지도 서비스.

### 3. 📸 현장 리뷰 시스템 (Geo-Verification)
- **50m 반경 인증**: 실제 위치 기반 작성 시스템.

### 4. 🏆 게임화된 성장 시스템 (Rank & Achievements)
- **30단계 랭크**: 비기너부터 레전드까지 승급.
- **50개 업적**: 다양한 카테고리의 업적 달성 시 골드 지급.

### 5. 📅 정기 자동 이벤트 (1st & 15th)
- 매달 1일과 15일, 24시간 한정 특별 이벤트.

### 6. 🚗 스마트 네비게이션 (Real-time Routing)
- 목적지까지의 실시간 최적 경로 안내.
