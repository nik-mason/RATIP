# Ratip 프로젝트 기능 명세서 📜

## 🗺️ Ratip: 나만의 지도를 그리는 여정
**Ratip**은 "남들이 좋다는 곳"이 아니라 **"내가 가본 곳, 내가 좋아할 곳"**에 집중하는 **개인화 위치 기록 & 게이미피케이션 플랫폼**입니다. 사용자가 직접 발을 딛고 기록한 장소들을 데이터화하여, 세상에 단 하나뿐인 나만의 지도를 완성해 나가는 여정을 지원합니다.

### ✨ 핵심 철학
- **진실성**: 실제 위치 인증을 통한 허위 정보 방지.
- **주관성**: 타인의 평가보다 나의 기준을 최우선으로 기록.
- **성취감**: 게임 요소를 결합하여 일상의 모든 이동을 재미로 승화.
- **지능형**: AI가 사용자의 패턴을 분석하여 최적의 경로와 장소 추천.

## ✨ 주요 기능 및 구현 계획

### 1. 로그인 (Login) 🔐
- **기능**: 일반 로그인 + 회원가입 + 생체 인식
- **구현 도구**: `supabase_flutter` (Auth), `local_auth` (Face ID) 👤✨

### 2. 지도 (Map) 🗺️
- **기능**: 사용자 위치 기반 대한민국 지도 표시 (Google Maps 스타일)
- **구현 도구**: `flutter_naver_map` 또는 `kakaomap_webview_flutter`, `geolocator` 📍

### 3. 위치 저장 및 리뷰 (Save Location & Review) 📸⭐
- **핵심 규칙**: 반드시 해당 장소 반경 **50m 이내**에 있어야만 리뷰 작성 가능 📍🚫 [인증 반경: 50m]
- **공유 설정**: 리뷰 작성 시 '공개하기' 또는 '나만 보기' 선택 (기본값: **나만 보기** 🔒)
- **노출 우선순위**: 내 리뷰가 항상 최상단 노출 🔝✨
- **구현 도구**: `geolocator`, `image_picker`, **Supabase Storage**

### 4. 업적 및 랭크 시스템 (Achievement & Rank System) 🏆💰
- **핵심 구조**: 업적 달성 ➔ **골드(Gold)** 획득 ➔ 골드로 **랭크(Rank)** 업그레이드 구매
- **구현 도구**: **Supabase DB**, `sqflite` (Local DB)
